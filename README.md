## Deploy to Vercel
1. Create a GitHub repo and push this folder as root.
2. On Vercel: Import GitHub repo, framework = Next.js.
3. Build command: `npm run build`, Output directory: (leave default `.next`).
4. Deploy.

## Run locally (optional)
```bash
npm install
npm run dev
```
